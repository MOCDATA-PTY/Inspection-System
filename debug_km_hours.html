🔍 KM and Hours Debug Helper

Step 1: Check Input Field Attributes

Right-click on the KM input field and select "Inspect Element". Look for these attributes:

<input type="number" class="form-control form-control-sm group-km-input" data-group-id="[SOME_GROUP_ID]" onchange="updateGroupKmTraveled(this)" onblur="updateGroupKmTraveled(this)">

⚠️ If any of these attributes are missing, that's the problem!

Step 2: Check Browser Console

Open browser console (F12) and run these commands:

// Check if the input field exists document.querySelector('.group-km-input') // Check the group ID document.querySelector('.group-km-input').getAttribute('data-group-id') // Check if the function exists typeof updateGroupKmTraveled // Check CSRF token getCSRFToken()

Step 3: Test the Function Manually

In the browser console, run:

// Get the input field const input = document.querySelector('.group-km-input'); // Set a test value input.value = '45'; // Call the function manually updateGroupKmTraveled(input);

✅ If this works, the function is fine. If not, there's a JavaScript error.

Step 4: Check Network Requests

1. Open Network tab in Developer Tools

2. Clear the network log

3. Enter a value in KM field and click away

4. Look for a request to /update-group-km-traveled/

5. Click on the request to see details

Expected Request: - Method: POST - URL: /update-group-km-traveled/ - Data: group_id, km_traveled, csrfmiddlewaretoken - Response: {"success": true, "message": "..."}

Step 5: Common Issues and Solutions

Issue 1: No console logs appear

❌ The JavaScript function isn't being called

Solution: Check if the input field has the correct event handlers

Issue 2: Console shows "updateGroupKmTraveled is not defined"

❌ The JavaScript function is missing

Solution: Refresh the page or check for JavaScript errors

Issue 3: Network request shows 403 Forbidden

❌ CSRF token issue or not logged in

Solution: Make sure you're logged in and CSRF token is valid

Issue 4: Network request shows 404 Not Found

❌ Endpoint doesn't exist

Solution: Check if the URL is correct

Issue 5: Network request shows 500 Internal Server Error

❌ Backend error

Solution: Check server logs for the error

Step 7: Expected Group ID Format

For "Boxer Superstore - Kwamashu 2" on 09/26/25, the group ID should be:

Boxer_Superstore_-_Kwamashu_2_20250926 or Boxer_Superstore___Kwamashu_2_20250926

⚠️ The exact format depends on how the client name is processed
